Stephanie Summer Tracker

这是一个专为 Stephanie 设计的暑假打卡与自我管理应用，旨在通过游戏化机制培养良好的学习与生活习惯。

核心功能
多语言支持：支持中、英、日三种语言界面。
任务打卡：每日追踪学习与锻炼任务。
智能结算系统：每日 18:00 后根据达标情况进行结算。
奖励机制：达标可积累礼品金或兑换娱乐奖励。

技术栈
开发框架：React + Vite
样式设计：Tailwind CSS
部署平台：Vercel
